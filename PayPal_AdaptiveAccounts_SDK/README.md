This C# Class Library project contains PayPal Adaptive Accounts Stubs. 

Prerequisites:
--------------
*	Visual Studio 2005 or higher

Stubs
--------
*	AdaptiveAccountsService.cs
	•	Service Version
	•	Service Name
	•	SDK Name
	•	SDK Version
	•	Coutries Supported
	•	Error information resulting from the service operation
*	PayPalAdaptiveAccountsModel.cs
	•	Default US culture info
	•	AckCodeType
	•	DetailLevelCodeType      
	•	Bank account type 
	•	API confirmation type 
    •	Business Types
	•	Subtype 
	•	Sales venue / store front type
	•	Stake holder role 
	•	SwitchMaestro, deprecated card type, use UKMaestro instead
